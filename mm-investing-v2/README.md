# MM Investing V2

A GitHub Pages ready static blog template inspired by the structure of large finance media company blogs.

## Files
- `index.html` homepage
- `post.html` single article page
- `category.html` category archive page
- `about.html` about page
- `404.html` fallback page
- `assets/css/styles.css` site styling
- `assets/js/data.js` demo data
- `assets/js/main.js` homepage rendering
- `assets/js/post.js` single post rendering
- `assets/js/category.js` category rendering
- `assets/img/*.svg` local placeholder art
- `robots.txt`
- `sitemap.xml`

## GitHub Pages
1. Create a new repository.
2. Upload every file while keeping the folder structure.
3. Open `Settings > Pages`.
4. Choose `Deploy from a branch`.
5. Select `main` and `/root`.
6. Save.

## Before publishing
- Replace `https://example.com` in `robots.txt` and `sitemap.xml` with your actual domain.
- Replace demo text in `assets/js/data.js`.
- Replace local SVG images with your own assets if needed.
